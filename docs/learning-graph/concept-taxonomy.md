# Concept Taxonomy

## Genetics: Analysis, Genomics, and Modern Inference

This taxonomy organizes 450 concepts into 12 categories for visualization and navigation.

| Category Name | TaxonomyID | Description |
|--------------|------------|-------------|
| Foundation Concepts | FOUND | Core prerequisite concepts that anchor the learning graph: genetic inference, genome organization, gene expression, and population genetics fundamentals |
| Probabilistic Reasoning | PROB | Bayesian reasoning, probability, statistical testing, and inference methods used in genetic analysis |
| Pedigree and Inheritance | PED | Pedigree analysis, inheritance patterns, penetrance, expressivity, and non-Mendelian modifications |
| Genome Structure | GSTR | Chromosome structure, chromatin organization, epigenetics, and genome-level features |
| Genetic Variation | GVAR | Types of genetic variation: SNPs, indels, CNVs, structural variants, repeats, and transposable elements |
| Mapping and Linkage | MAP | Genetic and physical mapping, linkage analysis, recombination, and gene discovery strategies |
| Quantitative Genetics | QUANT | Heritability, variance components, QTL mapping, GWAS, and polygenic traits |
| Population Genetics | POP | Allele frequencies, Hardy-Weinberg, selection, drift, gene flow, and population structure |
| Gene Regulation | REG | Transcription regulation, enhancers, noncoding RNA, chromatin state, and gene regulatory networks |
| Experimental Methods | EXP | Forward/reverse genetics, model organisms, mutagenesis, CRISPR, gene editing, and functional genomics |
| Genomics and Bioinformatics | BIOINFO | Sequencing technologies, alignment, variant calling, file formats, databases, and computational workflows |
| Human and Clinical Genetics | CLIN | Human disease genetics, genetic counseling, pharmacogenomics, cancer genetics, precision medicine, and gene therapy |
| Ethics and Society | ETHICS | Genetic privacy, discrimination, equity, gene editing ethics, DTC testing, and science communication |
| Frontier Topics | FRONT | CRISPR advancements, single-cell genomics, AI in genomics, pangenomics, and emerging methods |
